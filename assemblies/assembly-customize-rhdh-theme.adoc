= Customzing the appearance of your {product} instance
:context: assembly-customize-rhdh-theme

There are two default theme configurations available for {product}:

The {product} theme:: Default theme configurations to make your developer portal look like a standard {product} instance.

The Backstage theme:: Default theme configurations to make your developer portal look like a standard Backstage instance.

You can change or disable particular parameters in a default theme or create a fully customized theme by modifying the `app-config-rhdh.yaml` file.
